This review plugin helps you to interact with your users to improve your blog or content comments. Visitors submit reviews for different posts, products, events, or other content. You can display the reviews in an easier-to-read way using our WP Ultimate review plugin. It will also assist you to organize and style your WordPress site reviews more uniquely.
So, you can utilize the WP Ultimate Review plugin as a product review, customer review, user review, author review, rating plugin, wordpress review, post review, site review plugin, and more.

**WP Review Settings Features**

 - **Global settings-** The global settings option of a review plugin encompasses a range of customizable options. It allows you to tailor the plugin's behavior and appearance to your website's specific needs. Some options you will get under the settings are:
✔ Enable User review
✔ Enable Author review
✔ Restrict to Registered users only
✔ Require Admin approval
✔ Enable Author average
✔ Enable User average
✔ Per Post user review limit
✔ Per Post user review limit by
✔ Review Graph style
✔ Review Score limit
✔ Review Score input style
✔ Send to Administrator email
✔ Send to Author email

 - **Criteria-** If you want your product users to review the products according to ease of use, products' efficiency, your provided service, etc. then you can set criteria using the option. Besides products, you can set criteria for pages and post-wise also. 
 ✔ Product Criteria
 ✔ Page/Post Criteria
 
 - **Display Settings-** You can arrange the reviews to display beautifully using the option. The customization options available are : 
✔ Enable Review For
✔ Review Location
✔ Display Review with Comments
✔ Review Shown Per Page
✔ Review Form Settings
✔ Review Display Layout
✔ Average Rating Text
✔ Show Rating Text If

 - **reCaptcha-**  reCaptcha protects websites from spam and abuse. So, this is an option to help you verify that the submitted review or feedback is provided by a real person and not a computer program.
✔ Enable reCaptcha.

==FAQ==
How to use WP Ultimate Review?
Login to your WordPress dashboard, From the left menu click the WP Ultimate Review icon.

Will the plugin slow down the website?
No. WP Ultimate Review plugin is a well-coded, fast, and efficient tool. The developers do best practices of WordPress, and all our plugins are superfast.

How can I manage my reviews?
You can manage your reviews from the all reviews section of the plugin. There are options to approve, edit, and delete the reviews. In the settings section, you can customize the things to present your reviews based on your wishes.
